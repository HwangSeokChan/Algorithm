### 문제에 대한 생각
- location 값에 따른 종료
- Iterator 크기비교 : peek vs another
- max가 peek 일때 출력

### 다른 관점에서의 풀이
- Iterator 를 이용한 비교절차를 Arrays.sort 를 통해 _<U>***우선정렬***</U>
- 비교를 loop 가 아닌 if condition 으로 대체하여 <U>***반복루프 축소***</U> 로 성능향상
