### 다른관점에서의 풀이
- __정수론 정리__
  1. x를 홀수개의 연속된 자연수의 합으로 표현할 수 있는 경우
     - 1개
       - `b => 1*b` [단, b >= 1] 
     - 3개
       - `b-1, b, b+1 => 3*b` [단, b >= 2] 
     - 2a-1개
       - `b-(a-1), … b, …, b+(a-1) => (2a-1)*b` [단, b >= a] 
  2. x를 짝수개의 연속된 자연수의 합으로 표현할 수 있는 경우 
     - 2개
       - `d-1, d => 1*(2d-1)` [단, d >= 2] 
     - 4개
       - `d-2, d-1, d, d+1 => 2*(2d-1)` [단, d >= 3] 
     - 2d개
       - `d-c, …, d, …, d+(c-1) => c*(2d-1)` [단, d >= c+1] 
  3. 1.과 2.를 합쳐서 표현하면
     - `x = (2k-1)n` [단, n>=k이면 홀수개, n]
