### 문제에 대한 생각
- 소수 : 에라토스테네스의 체
    - 조합가능한 최대 수까지의 소수목록을 얻는다.
- 순열 : 완전방문탐색
    - 진수에 맞추어 자릿수를 증가시킨다.
- 중복 : Set
    - 수의 목록이 중복허용됨
    - 결과항목에 대한 중복 카운트를 방지함.
