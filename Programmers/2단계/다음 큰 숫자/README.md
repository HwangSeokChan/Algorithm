> - 조건
>   - 비트 1의 갯수가 같은 수
>   - 보다 큰수
>   - 조건을 만족하는 수중 가장 작은 수
> - 풀이
>   - 수의 증가
>   - 1의 수 확인

- psuedo code
  - Without Bit-Operator
  1. 반복문으로 첫 `LSB` 의 위치를 찾는다.
  2. 그보다 큰 `0` 의 위치를 찾는다.
