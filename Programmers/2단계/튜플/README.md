### 문제에 대한 관점
- 세부 문자열 (원소) 단위로 관리
    - split 활용
- 튜플의 순서를 확인
    - 세부 문자열의 크기순으로 정렬
- contain 은 equal 한 요소의 포함여부
    - 문자열 또는 객체는 값이 아닌 참조값으로 equal 여부판단
- Collection 별 성능차

### 다른 사람 풀이
- SolutionB
    - 갯수의 총합에 따른 순위
    - 람다식 풀이
    - Sort
        - Reverse
    - Map
        - mapToInt
- SolutionC
    - 갯수의 총합에 따른 순위
    - Regex
        - Pattern + Matcher
    - Map
        - Count
